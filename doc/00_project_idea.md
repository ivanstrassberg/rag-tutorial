# Идея проекта

**Учебный RAG на отзывах Amazon** — локальный pipeline для поиска и ответа по текстам отзывов покупателей.

**Автор:** Силин Иван Андреевич

## Проблема

Покупатели и аналитики не могут быстро найти релевантные отзывы среди тысяч записей.  
RAG позволяет задать вопрос на естественном языке и получить ответ **только из реальных отзывов** с указанием источника.

## Цель

Собрать учебный проект, который:

- показывает полный RAG-pipeline от CSV до Streamlit UI;
- работает на корпусе из **1500 отзывов** Amazon Fire HD 8;
- возвращает ответ **с источниками** (`doc_id`, score, текст фрагмента);
- явно **отказывается**, если релевантного контекста нет.

## Для кого

Студенты курса по Big Data / ML, изучающие retrieval-augmented generation.

## Данные

Источник — **Amazon Electronics Reviews** с Kaggle (`1429_1.csv`):

- открытые данные, легко скачать;
- структура: заголовок отзыва (`name`) + текст (`text`);
- 1500 записей — достаточно для оценки «отлично» по масштабу.

## Результат MVP

Приложение (Streamlit), куда можно задать вопрос, например:

> «Does this tablet have good battery life?»

Система находит подходящие фрагменты, формирует ответ и показывает, **из какого отзыва** взята информация.

## Критерий успеха

1. Pipeline работает end-to-end на 1500 отзывах.
2. 3 demo-вопроса дают ответ с источниками.
3. 1 negative-вопрос (`quantum physics relativity`) даёт отказ без галлюцинаций.
4. README позволяет воспроизвести запуск с нуля.
